Instructions:on cloud cd electronics/source_code and dont use profile

1. databricks configure (to configure profile from VSC)

2. databricks bundle validate  -t dev --profile oauth

3. databricks bundle deploy -t dev --profile oauth

4. To delete bundle: (if deleted need to deploy again)
   databricks bundle deployment unbind app-ui --target dev --profile oauth
   databricks bundle destroy -t dev --profile oauth 
   
5. To deploy app:
	databricks bundle run app-ui -t dev --profile oauth
